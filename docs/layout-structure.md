# 한입업무 — 레이아웃 구조도

## 화면 흐름
home (입력/분해) → today (실행체크) → report (광고 후 상세)

## 홈/실행 통합 화면
```
┌──────────────────────────────────────────┐
│ 상단 헤더: 한입업무 | N일 연속            │
├──────────────────────────────────────────┤
│ [오늘 할 일 한 줄 입력]                  │
│   예시 문구                              │
│   [5단계로 쪼개기]                       │
├──────────────────────────────────────────┤
│ [오늘 실행 플랜]                         │
│ ① 단계 제목 + 10분   [체크]              │
│ ② 단계 제목 + 15분   [체크]              │
│ ③ 단계 제목 + 15분   [체크]              │
│ ④ 단계 제목 + 10분   [체크]              │
│ ⑤ 단계 제목 + 5분    [체크]              │
├──────────────────────────────────────────┤
│ [부가 기능]                              │
│ 주간 리포트 상세 보기  [AD]              │
│ 스트릭 복구            [AD]              │
├──────────────────────────────────────────┤
│ [요약] 현재 스트릭 | 최고 기록 | 주간 평균 │
└──────────────────────────────────────────┘
```

## 리포트 바텀시트
```
┌──────────────────────────────────────────┐
│ 주간 리포트                              │
│ 완료 일수 / 평균 완료율                  │
│ ---------------------------------------- │
│ 날짜별 기록 카드 목록                    │
│ - YYYY-MM-DD / 입력 문장 / 완료 단계     │
└──────────────────────────────────────────┘
```

## 차별화 체크리스트 (vs 하루모아/오늘 한 줄)

| 항목 | 한입업무 | 기존 앱 | 차별화 |
|---|---|---|---|
| 핵심 입력 | 1문장 할 일 | 날짜/감정/기록 | ✅ |
| 결과 구조 | 5단계 실행 분해 | 카운트다운/일기 | ✅ |
| 절대 규칙 | 단계당 15분 제한 | 제한 없음 | ✅ |
| 광고 포인트 | 리포트/복구 | 수정/부가 분석 | ✅ |
| 반복 사용 이유 | 당일 실행 플랜 | 기록/일정 확인 | ✅ |
| 화면 흐름 | 입력→실행체크 중심 | 기록/캘린더 중심 | ✅ |
| 고유 컴포넌트 | 5단계 제약 엔진 | 없음 | ✅ |

합계: 7/7
